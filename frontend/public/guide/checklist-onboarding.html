Semaine 1 — Priorite haute (minimum viable)

1
Donnees critiques pour demarrer
Sans ces donnees, la plateforme ne peut pas etre utilisee en production.
ActionResponsableMenuDetail
Verifier les adresses clients Comptable Referentiels → Clients → detail Adresse facturation complete pour K+N, Geodis, DB Schenker, Central Express
Saisir les tarifs par client Comptable Finance → Tarifs → "Nouveau tarif" Montant HT par type de prestation pour chaque client. Necessaire pour la facturation.
Verifier le compte bancaire Comptable Parametrage → Parametres → Banque Remplacer l'IBAN de demo par le vrai IBAN de l'entreprise
Verifier les infos entreprise Admin Parametrage → Parametres → Entreprise SIREN, SIRET, TVA intracommunautaire, adresse, telephone, email
Scanner les documents expires RH Referentiels → Conducteurs → detail → Conformite 9 documents expires detectes (ADR, carte identite, visite medicale). Scanner et uploader les versions a jour.
Verifier les montants des tournees Dirigeant Exploitation → Tournees → detail Montant vente HT et achat HT par tournee pour le calcul de rentabilite

Semaine 2 — Donnees operationnelles

2
Completer pour l'exploitation quotidienne
ActionResponsableMenuDetail
Ajouter les contacts clients Exploitation Referentiels → Clients → detail → Contacts Nom, email, telephone du contact principal et facturation par client
Ajouter les adresses de livraison Exploitation Referentiels → Clients → detail → Adresses Adresses de chargement et livraison pour chaque client/site
Saisir les salaires conducteurs RH Referentiels → Conducteurs → detail → Contrat Salaire base mensuel et taux horaire pour le calcul de rentabilite
Saisir les couts vehicules Flotte Referentiels → Vehicules → detail → Couts Gasoil, assurance, entretien, reparation, peage du mois en cours
Configurer les arrets des tournees Exploitation Exploitation → Tournees → detail → Arrets Points de chargement et livraison par tournee pour le CMR
Generer les executions du mois Exploitation Exploitation → Tournees → detail → Executions → Generer Selectionner la periode (ex: 01/05 au 31/05) et generer

Semaine 3 — Completude et conformite

3
Objectif : 100% de conformite
ActionResponsableMenuDetail
Scanner TOUS les documents conducteurs RH Referentiels → Conducteurs → detail → Conformite Permis, FIMO, FCO, carte conducteur, visite medicale, carte identite, ADR
Scanner TOUS les documents vehicules Flotte Referentiels → Vehicules → detail → Conformite Carte grise, controle technique, assurance, attestation capacite
Ajouter les sous-traitants Exploitation Referentiels → Sous-traitants Si SAF utilise des sous-traitants pour certaines tournees
Verifier le tableau de bord conformite Admin Pilotage → Conformite Objectif : 100% — tous les documents valides pour tous les conducteurs et vehicules
Planifier les maintenances Flotte Flotte → Maintenance → "Nouvelle maintenance" Controles techniques, vidanges, tachygraphes a planifier
Former les utilisateurs Admin Guide utilisateur S'assurer que chaque profil connait ses actions (voir guide, etape 10)
