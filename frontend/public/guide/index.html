Qui : Administrateur de l'entreprise Où : Page de connexion

3
Installer les données de référence
Qui : Administrateur de l'entreprise Où : Exploitation → Configuration

Cette étape installe automatiquement toutes les données de base françaises.

  1. Dans le menu latéral, ouvrir « Exploitation »
  2. Cliquer sur « Configuration »
  3. La page affiche une checklist de 6 éléments
  4. Dans la section « Démo rapide », cliquer sur « Installer les données démo »
  5. Attendre la fin de l'installation (quelques secondes)
Page Configuration initiale — Exploitation → Configuration
Ce qui est installé automatiquement :
  • Types de documents conducteur : Permis de conduire, FIMO, FCO, Visite médicale, Carte conducteur, ADR, Carte d'identité
  • Types de documents véhicule : Carte grise, Contrôle technique, Assurance, Attestation de capacité
  • Variables de paie : Heures normales, HS 25%, HS 50%, Heures nuit, Primes, Indemnités, Absences
  • Taux de TVA : 20%, 10%, 5.5%, 2.1%
  • Données de démo : 3 clients, 3 conducteurs, 3 véhicules, 1 sous-traitant

Ajouter des utilisateurs
Qui : Administrateur de l'entreprise

Ajouter les collaborateurs avec le rôle adapté à leur fonction :

Profil 1 — Admin / Dirigeant

Rôle : admin — Accès complet à tous les modules
  • Créer et gérer les missions de transport
  • Gérer les référentiels (clients, conducteurs, véhicules, sous-traitants)
  • Facturation : créer des factures, avoirs, gérer les tarifs
  • Gérer la flotte : maintenance, sinistres, tableau de bord
  • Consulter les rapports et KPI
  • Configurer les paramètres de l'entreprise (société, banque, TVA)
  • Gérer les utilisateurs et leurs rôles
  • Consulter le journal d'audit
  • Gérer la conformité documentaire

Sections du menu : Exploitation, Référentiels, Finance, Flotte, Pilotage, Paramétrage

Profil 2 — Exploitant / Dispatcher

Rôle : exploitation — Gestion opérationnelle des missions
  • Créer, modifier et suivre les missions (dossiers transport)
  • Affecter conducteurs et véhicules aux missions
  • Gérer les points de livraison et les marchandises
  • Suivre les preuves de livraison (POD)
  • Gérer les litiges
  • Consulter les référentiels (clients, conducteurs, véhicules)
  • Mettre à jour les données référentiels
  • Gérer les tâches opérationnelles
  • Télécharger des documents de conformité

Sections du menu : Exploitation, Référentiels

Vue du menu pour un Exploitant

Profil 3 — Comptable

Rôle : compta — Gestion financière et facturation
  • Créer et valider les factures
  • Créer et valider les avoirs (credit notes)
  • Gérer les règles de tarification
  • Scanner et traiter les documents par OCR
  • Valider les factures extraites par OCR
  • Consulter les missions (lecture seule)
  • Gérer les relances et le recouvrement
  • Consulter les rapports financiers
  • Gérer les factures fournisseurs

Sections du menu : Exploitation, Finance, Pilotage

Vue du menu pour un Comptable

Profil 4 — Responsable RH / Paie

Rôle : rh_paie — Gestion des ressources humaines et paie
  • Saisir et gérer les variables de paie (heures, primes, absences)
  • Importer / exporter les données paie (format SILAE)
  • Soumettre et valider les périodes de paie
  • Gérer les fiches conducteurs (données personnelles, contrat)
  • Télécharger et suivre les documents conducteurs (permis, FIMO, visite médicale...)
  • Suivre les alertes de conformité conducteurs
  • Gérer les tâches RH

Sections du menu : Exploitation, Référentiels, Finance

Profil 5 — Gestionnaire de Flotte

Rôle : flotte — Gestion des véhicules et maintenance
  • Consulter le tableau de bord flotte (conformité, alertes)
  • Planifier et suivre les maintenances (préventives et curatives)
  • Déclarer et gérer les sinistres
  • Gérer les fiches véhicules (carte grise, contrôle technique, assurance)
  • Suivre les coûts véhicules
  • Télécharger les documents véhicules
  • Suivre les alertes de conformité véhicules

Sections du menu : Référentiels, Flotte

Vue du menu pour un Gestionnaire de Flotte

Profil 6 — Auditeur / Lecture seule

Rôle : lecture_seule — Consultation de tous les modules sans modification

  • Consulter le journal d'audit
  • Consulter la conformité documentaire

9
Utilisation quotidienne — Tournées et Missions
Qui : Exploitant / Dispatcher

Le flux de travail quotidien repose sur les tournées. Les missions sont générées automatiquement.

9.1 — Créer une tournée modèle (une seule fois par route)

Menu : Exploitation → Tournées
  1. Cliquer "Nouveau modèle"
  2. Remplir : Code (ex: 1406), Libellé (ex: K+N Epône), Client, Site
  3. Choisir la récurrence : Lun-Ven (par défaut), Lun-Sam, Quotidienne, etc.
  4. Affecter le conducteur par défaut et le véhicule par défaut
  5. Saisir le montant vente HT (revenu journalier) et montant achat HT (coût journalier)
  6. Enregistrer

Résultat : Le modèle apparaît dans la liste des tournées avec son conducteur et véhicule affectés.

9.2 — Générer les exécutions du mois

Menu : Exploitation → Tournées → cliquer sur un modèle → onglet "Exécutions"
  1. Saisir la période : Date début et Date fin (ex: 01/04 → 30/04)
  2. Cocher "Créer les missions automatiquement"
  3. Cliquer "Générer"

Résultat : Le système crée automatiquement :

  • 1 exécution par jour ouvré (selon la récurrence)
  • 1 mission par exécution (avec conducteur, véhicule et montants hérités du modèle)

Exemple : Pour avril 2026 (Lun-Ven) = 22 exécutions + 22 missions générées en un clic.

9.3 — Gestion quotidienne

Menu : Exploitation → Exécutions
  • Voir les tournées du jour (filtrer par date)
  • Si un conducteur est absent → cliquer l'exécution → modifier le conducteur
  • Si un véhicule est en panne → cliquer l'exécution → modifier le véhicule
Menu : Exploitation → Missions
  • Les missions progressent : PLANIFIÉE → AFFECTÉE → EN COURS → LIVRÉE → CLÔTURÉE → FACTURÉE
  • Le dispatcher change le statut au fur et à mesure de la journée
  • Chaque mission affiche son modèle source et son exécution associée
Ce que l'utilisateur ne fait PAS manuellement :
  • Créer des missions individuelles — elles sont générées depuis les tournées
  • Affecter conducteur/véhicule par mission — hérité de l'exécution
  • Calculer les revenus/coûts — hérités des montants du modèle
  • Vérifier la conformité — le scan quotidien tourne automatiquement

Conformité et alertes documentaires
Qui : RH, Fleet Manager, Admin

Comment fonctionnent les alertes

Le système vérifie automatiquement chaque jour les dates d'expiration de tous les documents (permis, FIMO, carte grise, contrôle technique, etc.).

Des alertes sont créées à 90, 60, 30, 15 et 7 jours avant l'expiration.

Où voir les alertes

EmplacementCe qui apparaît
Icône cloche (barre latérale)Badge rouge avec nombre de notifications
Conducteurs / Véhicules (liste)Ligne rouge = BLOQUANT, orange = À RÉGULARISER
Conducteur / Véhicule (détail)Bannière rouge/orange en haut de page
ConformitéDashboard avec taux de conformité par entité
Flotte → Tableau de bordWidget "Expirations à venir (60j)"

Comment corriger une alerte

  1. Aller dans Référentiels → Conducteurs (ou Véhicules)
  2. Cliquer sur l'entité en rouge/orange
  3. Aller à l'onglet Conformité
  4. Voir les documents manquants ou expirés
  5. Cliquer "Télécharger" pour uploader le nouveau document
  6. Remplir la date d'expiration du nouveau document
  7. L'alerte disparaîtra au prochain scan quotidien
