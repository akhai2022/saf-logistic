Guide de Création et Configuration d'une Entreprise

  2. Se connecter en tant qu'administrateur de l'entreprise
  3. Installer les données de référence (données démo)

  5. Créer les référentiels métier
  6. Créer les règles de tarification
  7. Ajouter des utilisateurs
  8. Vérifier que tout est prêt
1
Créer une nouvelle entreprise
Qui : Super Administrateur Où : Administration → Entreprises
  1. Se connecter sur https://saf.dataforgeai.fr/login avec le compte Super Admin
  2. Dans le menu latéral, cliquer sur Administration → Entreprises
  3. Cliquer sur le bouton « Créer une entreprise »
  4. Remplir le formulaire :

  • L'email et le mot de passe de l'administrateur

2
Se connecter en tant qu'administrateur
Qui : Administrateur de l'entreprise Où : Page de connexion
  1. Accéder à https://saf.dataforgeai.fr/login
  2. Remplir les 3 champs :
    • Email : l'email fourni par le Super Admin
    • Mot de passe : le mot de passe fourni
    • Tenant ID : le UUID fourni (ex: 7c74c775-5c14-4aa1-9ef1-1cce6c5a1fa3)
  3. Cliquer sur « Se connecter »
Formulaire de connexion avec les identifiants
3
Installer les données de référence
Qui : Administrateur de l'entreprise Où : Exploitation → Configuration

Cette étape installe automatiquement toutes les données de base françaises.

  1. Dans le menu latéral, ouvrir « Exploitation »
  2. Cliquer sur « Configuration »
  3. La page affiche une checklist de 6 éléments
  4. Dans la section « Démo rapide », cliquer sur « Installer les données démo »
  5. Attendre la fin de l'installation (quelques secondes)
Page Configuration initiale — Exploitation → Configuration
Ce qui est installé automatiquement :
  • Types de documents conducteur : Permis de conduire, FIMO, FCO, Visite médicale, Carte conducteur, ADR, Carte d'identité
  • Types de documents véhicule : Carte grise, Contrôle technique, Assurance, Attestation de capacité
  • Variables de paie : Heures normales, HS 25%, HS 50%, Heures nuit, Primes, Indemnités, Absences
  • Taux de TVA : 20%, 10%, 5.5%, 2.1%
  • Données de démo : 3 clients, 3 conducteurs, 3 véhicules, 1 sous-traitant

Configurer les paramètres de l'entreprise
Qui : Administrateur de l'entreprise Où : Paramétrage → Paramètres

4.1 — Informations de l'entreprise

Onglet « Entreprise » — Remplir :

TVA IntracommunautaireOui

Téléphone, EmailRecommandé

Paramétrage → Paramètres → Onglet Entreprise

4.2 — Comptes bancaires

Obligatoire pour générer des factures. Ajouter au moins 1 compte bancaire avec IBAN.
Paramétrage → Paramètres → Onglet Banque

4.3 — Vérifier les taux de TVA

Si les données démo ont été installées, les 4 taux français sont déjà configurés. Vérifier que le taux 20% est par défaut.

Paramétrage → Paramètres → Onglet TVA

7
Ajouter des utilisateurs
Qui : Administrateur de l'entreprise

Ajouter les collaborateurs avec le rôle adapté à leur fonction :

admin

Dirigeant — accès complet

exploitation

Exploitant — missions, planning, litiges

compta

Comptable — factures, tarifs, OCR

rh_paie

RH — paie, conducteurs, documents

flotte

Fleet manager — véhicules, maintenance

lecture_seule

Auditeur — lecture seule

Vue du menu pour un Exploitant
8
Vérifier que tout est prêt
Où : Exploitation → Configuration

Retourner sur la page Configuration et vérifier que les 6 éléments sont cochés en vert :

✓ Clients
✓ Conducteurs
✓ Véhicules
✓ Types de documents FR
✓ Règles de tarification
✓ Variables de paie FR
Configuration terminée ! L'entreprise est prête. Vous pouvez maintenant créer des missions, affecter des conducteurs, générer des factures et gérer votre flotte.
